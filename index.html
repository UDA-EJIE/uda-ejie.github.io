Utilidades de Desarrollo de Aplicaciones

View project on GitHub

UDA es el conjunto de utilidades, herramientas, librerías, plugins, guías, y recomendaciones funcionales y técnicas que permiten acelerar el proceso de desarrollo de sistemas software con tecnología Java.

🚀 Empezar Ahora 🎮 Ver Demo

🎯 Objetivos y Características

El espíritu del proyecto es aumentar la productividad del desarrollador, sin coartar su imaginación ni libertad para crear software. UDA ayuda en las tareas repetitivas y de poco valor añadido, pero también persigue otros fines:

📚 Curva de Aprendizaje Mínima

No pretende ser un framework más, adopta y explota los ya consolidados y aprovecha el conocimiento compartido.

🌐 Patrones de Usabilidad Web

Aprovecha las mejoras de tecnologías RIA - Ajax para escenarios habituales.

⚡ Generación Automática de Interfaces

Crea automáticamente interfaces de mantenimiento (acciones CRUD) y componentes comunes.

🏗️ Generación de Código

Crea código base para aplicaciones con arquitectura en capas y paradigma MVC.

🔧 Flexible y Configurable

Adapta el código generado a las necesidades organizacionales (estilos, estándares, etc.).

♿ Aplicaciones Accesibles

Cumple con recomendaciones WCAG2.0 y WAI-ARIA.
